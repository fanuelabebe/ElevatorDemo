# ElevatorDemo

## Android Elevator Simulator
In this app you will be able to simulate the basic function of an Elevator

### The basics
- Elevator moves from each floor to another with a timed manner and displays colors based on floor status
- Accepts calls from each floor from users and operates accordingly
- Elevator has a maintnance count which will break down if this count is passed
- Elevator has a repair button to repair it and logs the broken and repair times in SQLite
